# CamDetector 启动说明

## 环境要求

- Node.js 20 或更高版本
- npm
- 现代浏览器

## 本地启动前端

```bash
cd frontend
npm install
npm run dev
```

启动后打开终端输出的本地地址。

## 构建静态站点

```bash
cd frontend
npm run build
```

构建产物位于：

```text
frontend/dist
```

## GitHub Pages 部署

仓库已配置 GitHub Actions：

```text
.github/workflows/deploy.yml
```

推送到 `main` 后会自动执行：

1. 安装前端依赖
2. 构建 Vite 静态页面
3. 上传 `frontend/dist`
4. 发布到 GitHub Pages

在 GitHub 仓库设置中，Pages Source 选择 `GitHub Actions` 即可。

## 演示路径

前端使用 hash 路由，适合 GitHub Pages 子路径部署：

- `/`：平台首页
- `#/detection`：检测控制台
- `#/localization`：定位工作台
- `#/mobile`：移动端四步流程
- `#/algorithms`：高级工具
- `#/evaluation`：风险报告
- `#/repository`：系统设置与边界说明

## 注意事项

本项目只使用合成数据进行安全演示，不会访问真实无线网卡、不会抓取真实 802.11 流量，也不会执行破解或设备控制操作。
