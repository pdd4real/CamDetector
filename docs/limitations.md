# Limitations

This repository is a controlled demonstration, not a deployable surveillance or network security tool.

- No real packet capture is performed.
- No Wi-Fi monitor mode is enabled.
- No Aircrack-ng command or password cracking workflow is included.
- No Android NDK binary is included.
- No camera is accessed, scanned, or exploited.
- The SVM and CUSUM modules use deterministic demo parameters.
- All packet frames, MAC addresses, and evaluation metrics are synthetic.

These boundaries keep the project useful for explaining the CamDetector concept while avoiding operational network intrusion behavior.
