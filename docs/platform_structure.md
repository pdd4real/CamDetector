# CamDetector 平台结构说明

CamDetector 的展示结构参考完整安全检测平台的组织方式，将“项目介绍”和“功能工作台”拆成清晰模块。

## 页面模块

| 模块 | 说明 |
| --- | --- |
| 首页 | 项目定位、实时感知面板、能力矩阵、数据指标和技术架构 |
| 检测控制台 | 数据采集、帧解析、特征向量、SVM 分类和图表联动 |
| 定位工作台 | CUSUM 趋势、雷达方向、房间标注和距离估计 |
| 移动端流程 | 四步手机交互，用于录屏或嵌入 PPT |
| 高级工具 | 展示算法流水线与关键中间状态 |
| 风险报告 | 展示 TPR/TNR、样本规模、动作时长和品牌评估 |
| 系统设置 | 工程结构、安全边界和演示限制 |

## 仓库模块

| 目录 | 说明 |
| --- | --- |
| `frontend/` | 静态前端展示站 |
| `algorithms/` | 可读的模拟算法实现 |
| `samples/` | 合成数据与评估样本 |
| `docs/` | 架构、流程、算法与限制 |
| `tests/` | 基础 smoke test |

## 设计边界

本项目不提供真实无线抓包、破解或设备访问能力。所有可视化结果均由合成样本驱动，用于说明算法思想和产品交互流程。
